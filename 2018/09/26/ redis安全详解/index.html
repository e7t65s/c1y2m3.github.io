    在web目录下写入webshell
    写入ssh公钥，获取操作系统权限

    将本机的公钥作为value，key值随意，然后通过修改数据库默认路径和缓冲文件，把缓冲文件写在文件里，等于在服务端/root/.ssh下生成授权的key.

    客户端生成Key: ssh-keygen -t rsa

    目标主机查看写入keys内容:

    cat /root/.ssh/authorized_keys
